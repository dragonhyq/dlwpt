A good simple tutorial for absolute beginners: [Jupyter Notebook for Beginners: A Tutorial](https://www.dataquest.io/blog/jupyter-notebook-tutorial/)


Keyboard shortcut:

| shortcut | Meaning |
|:--- | :--- |
| `Ctrl + Enter` | run current cell |
| a | insert cell above |
| b | insert cell below |
| d + d | delete cell | 
| m | convert cell to Markdown |
